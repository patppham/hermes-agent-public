# Systematic runtime QA

Use this skill to test a human-facing dashboard or local workflow as a user,
not just as a unit-test author. Define the route and expected outcome, exercise
the smallest representative flow, capture evidence for failures, and classify
the issue as correctness, usability, delivery, or environment.

Keep screenshots, session state, authenticated URLs, and personal fixture data
out of the public repository. Report reproducible steps and expected versus
actual behavior without copying private records.
