# Reference grounding

Use this skill when a response depends on a local reference corpus, a current
authority, or a safety-sensitive claim.

Apply the private source hierarchy: use a configured local reference for stable
background, then verify clinical, legal, age-specific, jurisdiction-specific,
or time-sensitive claims against a current authoritative source. Cite the
relevant section or source near the claim. If sources disagree, prefer the
current authoritative guidance and say that the sources differ.

Do not expose raw corpus content, private document locations, credentials, or
unrelated records. Retrieval failure is uncertainty, not permission to guess.
