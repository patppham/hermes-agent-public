# Annual reminder

Use the private schedule adapter's authorized reminder template and recipient.
Verify that the event is configured, current, and not already handled. Keep the
message short and useful, and include one next action only when the template
requires it.

The public repository contains no names, dates, recipients, or message text.
If any required private value is missing, return `NO_OUTPUT`. Do not infer a
recipient, disclose the reminder in a shared channel, or send without explicit
approval.
